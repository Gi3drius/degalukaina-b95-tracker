How refresh works

GitHub Pages is static, so the browser cannot run the Python scraper or update the database. The Refresh button reloads the latest committed docs/data/fuel_prices.json file. To update prices automatically, run the included GitHub Actions workflow or run the scraper locally, export JSON, and push the changed data file.
