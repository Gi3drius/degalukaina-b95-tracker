How refresh works

GitHub Pages is static, so the browser cannot securely run the Python scraper or write to this repository. Reload latest data only reloads the latest committed docs/data/fuel_prices.json file.

To force a new scrape, open the GitHub Actions workflow and run Update fuel prices manually. A fully automatic website button would need a small backend/proxy with a protected GitHub token; putting that token directly in this static page would expose repository write access to every visitor.
